# [Play game here](https://cazdev.itch.io/not-so-bright-world)

This game includes 4 levels as well as a final boos fight.  The game can be played through a web browser using unity Web-GL. 

Controls:
- W - Jump
- A - Move left
- S - Crouch
- D - Move right
- Space - Jump
- O - Fire Range Weapon
- K - Melee Attack
- ESC - Pause
